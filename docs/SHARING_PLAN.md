# GitHub・共有サイト・Steam配布案

## まずGitHub

最初はGitHubが一番現実的です。

- 無料で公開できる
- `app/` と `micro_ai_shelf/` を分けて管理できる
- `agents/*.json` をPull Requestで追加できる
- Issuesで不具合やAIパック提案を集められる
- GitHub Pagesで簡単なカタログサイトに発展できる

## 極小AI共有サイト

次の段階では、極小AIを検索・追加できる共有サイトを作ると面白いです。

評価は人気順だけでなく、軽さ、平均応答時間、失敗率、安全チェック、対応RAM目安を重視します。

## Steam

Steamは完成アプリの配布先としてはありです。  
ただし、極小AIパック共有の中心にはGitHubや専用サイトの方が向いています。

おすすめ順:

1. GitHubで公開
2. GitHub Pagesか小さいWebサイトでAI棚カタログ
3. Windowsアプリとして整ったらSteamを検討
